# Release Notes - TMI Publication Package 0.2

## Included publication units

- TMI Core 0.2: automatic interfaces and primary interface axiom.
- TMI-Physics 2.2: automatically structured interface fields.
- Computational Validation Report v2: Tests 1-11.

## Main additions in 0.2

- Automaticity is treated as a definitional property of valid interfaces.
- The primary interface axiom is connected to the physical layer.
- The scalar field chi_I is preserved as an amplitude projection of Xi_I^auto.
- Tests 7-11 extend the computational validation with robustness, failure boundaries and safety recovery.

## Current limitations

- Core and Physics documents are currently packaged in Russian.
- Computational report v2 is currently packaged in Russian.
- Test scripts are bundled mainly through test archives; a unified reproducibility runner is still the next step.
